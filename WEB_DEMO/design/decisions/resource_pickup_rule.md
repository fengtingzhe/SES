# 决策记录：辉石拾取规则

## 决策结论

WEB_DEMO 后续采用 GPT_DEMO 的辉石拾取规则：

```text
自然掉落 / 地图生成的辉石采用靠近自动拾取。
```

也就是说，玩家不需要按 Space 拾取自然辉石。

---

## 规则说明

### 必须继承

```text
1. 玩家靠近自然辉石后自动拾取。
2. 拾取后辉石从地图上移除。
3. 玩家辉石库存增加对应数量。
4. 自动拾取属于探索与资源收集的基础手感。
```

### Space 的职责

Space 不再用于自然辉石拾取。

Space 仍然用于：

```text
1. 面前格优先互动。
2. 工人派工类互动。
3. 修桥、砍树、建营地等目标互动。
4. 无互动目标时放置 1 个辉石。
```

### 临时放置辉石

玩家主动放置的临时辉石仍然保留：

```text
1. 无互动目标时 Space 放置。
2. 消耗玩家 1 个辉石。
3. 拥有生命周期。
4. 后续用于黑影诱敌。
```

---

## 对 v0.2-refactor 的影响

v0.2-refactor 中曾按任务卡实现“面向辉石按 Space 拾取”。

该行为在后续版本中应修正为：

```text
靠近自动拾取自然辉石。
```

这不是有意重设计，而是对 GPT_DEMO 规则的回归。

---

## 对 v0.3-refactor 的要求

v0.3-refactor 开始前，应先完成该修正：

```text
1. 从 InteractionSystem 中移除自然辉石的 Space 拾取职责。
2. 在 ResourceSystem 或独立 PickupSystem 中实现靠近自动拾取。
3. HUD 和提示文案同步调整。
4. 更新规则继承审计文档。
```
