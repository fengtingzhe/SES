# Changelog

## v0.1-refactor

状态：WEB_DEMO 阶段重置

内容：
- 废弃此前 WEB_DEMO v0.2～v0.6 的 Codex 重新开发路线。
- 删除旧 WEB_DEMO 运行工程内容。
- 重新定义 WEB_DEMO 阶段目标为"理解并重构 GPT_DEMO"。
- 明确 GPT_DEMO 是玩法规则基线，WEB_DEMO 是工程化重构版本。
- 新增 WEB_DEMO 重构策略文档。
- 新增 GPT_DEMO 规则基线模板。
- 新增 GPT_DEMO 到 WEB_DEMO 的规则继承审计模板。
- 当前阶段不包含可运行游戏工程。

## 已废弃路线

以下旧版本路线已废弃，不再作为主线继续开发：

- WEB_DEMO v0.2 核心循环迁移
- WEB_DEMO v0.3 GameConfig 基础配置集中化
- WEB_DEMO v0.4 夜晚黑影压力系统
- WEB_DEMO v0.5 围墙 / 弓箭手 / 防御系统
- WEB_DEMO v0.6 矿山 / 流民 / 人口补给系统

废弃原因：
这些版本是重新开发一套类似 GPT_DEMO 的功能，而不是以 GPT_DEMO 为规则基线进行工程化重构。
